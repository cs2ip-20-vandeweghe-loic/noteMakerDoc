## Initialisation

### Créer un environnement virtuel

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### Mettre à jour les dépendances

```bash
pip freeze > requirements.txt
```

### Lancer Zensical en local

```bash
zensical serve
```
